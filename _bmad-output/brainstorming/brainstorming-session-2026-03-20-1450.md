# Brainstorming Session — Programme Ambassadeur EnfantDisparu.bf

**Facilitateur:** Swabo
**Date:** 2026-03-20

---

## Contexte

**EnfantDisparu.bf** est une association à but non lucratif. Le système fonctionne en automatique (alertes publiées → diffusion Facebook, TikTok, etc.). Swabo reste anonyme avec son agent IA qui gère les réseaux.

**Objectif Ambassadeurs:** Créer une armée de terrain pour faire grandir le réseau et rendre enfantdisparu.bf visible partout sur Facebook au Burkina Faso.

---

## Décision Validée

✅ **Nom retenu : AMBASSADEUR**

---

## Missions Ambassadeur (5 piliers)

| Mission | Action concrète |
|---------|-----------------|
| **1. Partager** | Diffuser les alertes dans ses groupes WhatsApp et son quartier |
| **2. Capturer** | Ramener les annonces Facebook "sauvages" vers le site |
| **3. Convertir** | Convaincre les parents de poster via enfantdisparu.bf |
| **4. Recruter** | Inviter d'autres Ambassadeurs |
| **5. Activer** | Convaincre les gens de visiter le site ET accepter les notifications OneSignal |

---

## Système de Tracking Notifications

- Chaque Ambassadeur a un **lien unique** : `enfantdisparu.bf/?ref=AMB-xxxx`
- Quand quelqu'un accepte les notifs via ce lien → **attribué à l'Ambassadeur**
- Stat visible sur son dashboard : "X personnes ont activé les alertes grâce à toi"
